# Model Context Protocol (MCP)

The Model Context Protocol (MCP) connects dBocl to LLMs and AI applications, enabling seamless AI-powered interactions with our platform. Our MCP server is automatically generated from our documentation, making it easy for AI applications to understand and interact with dBocl.

## Quick Start

Install the dBocl MCP server with a single command:

```bash
npx mint-mcp add dbocl
```
